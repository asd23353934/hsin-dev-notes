## 紀錄模板

```markdown
### 問題標題
- **適用版本**：PostgreSQL 14-16（**必填**，標明此問題存在/解法成立的版本範圍；上游已修復則註記「v17 起已修復」）
- **日期**：YYYY-MM-DD
- **環境**：實際當下踩到時的精確版本（例：PostgreSQL 16.1 / 連線端 pg 8.11.3 / Node 22.5）
- **問題**：症狀（錯誤訊息、行為）
- **原因**：根因
- **解法**：步驟或程式碼
- **參考**：issue / 文件連結（可選）
```

> **「適用版本」與「環境」差別**：
> - **適用版本** = 此問題 / 解法在哪些 PG 大版本範圍成立（給未來查找用）
> - **環境** = 實際踩到時的精確版本（給可重現用）
